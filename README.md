**INSTALL GUIDE**
--------------------------------------------------
1. Clone repo
2. Copy all bspwm/bspwm-rose-pine/config files in ~/.config
3. Copy xinitrc file in yours
4. Install picom, polybar, feh, alacritty, bspwm, rofi, neovim, sxhkd and all there dependencies
5. Enjoy!
