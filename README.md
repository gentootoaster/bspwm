**INSTALL GUIDE**
--------------------------------------------------
1. Clone repo
2. Copy all bspwm/bspwm-rose-pine/config files in ~/.confif
3. Install picom, feh, alacritty, bspwm, rofi, neovim, sxhkd and all there dependencies
4. Enjoy!
